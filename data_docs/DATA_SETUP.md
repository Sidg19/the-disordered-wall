# Data setup


Download the following public datasets and place them locally before running the notebook:

- [ProteinGym download page](https://proteingym.org/download)
  - `DMS_substitutions.csv`
  - `DMS_ProteinGym_substitutions/`
  - `ProteinGym_AF2_structures/` *(optional for this final notebook; not required for the final successful results)*

- [DisProt annotated proteins download page](https://disprot.org/download)
  - `DisProt_release_2025_12.tsv`

Then update the paths in `notebooks/disordered_wall_pipeline_final.ipynb`.
